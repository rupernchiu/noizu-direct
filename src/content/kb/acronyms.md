## Payment rails

| Acronym  | Country | Meaning |
|----------|---------|---------|
| FPX      | MY      | Financial Process Exchange — bank account direct debit |
| DuitNow  | MY      | Real-time payment + QR rail |
| TNG      | MY      | Touch 'n Go eWallet |
| BOOST    | MY      | Boost eWallet |
| FAST     | SG      | Fast And Secure Transfers |
| PayNow   | SG      | Mobile-number/UEN payment |
| GCash    | PH      | Globe Telecom eWallet |
| InstaPay | PH      | BSP-operated real-time payment |
| BI-FAST  | ID      | Bank Indonesia Fast Payment system |
| PromptPay| TH      | National real-time payment rail |
| GrabPay  | regional| Grab eWallet (MY/SG/PH/ID/TH) |
| SWIFT    | global  | Society for Worldwide Interbank Financial Telecommunication — bank wire |
| 3DS      | global  | 3-D Secure — issuer-side card auth (Verified by Visa, etc.) |
| BIN      | global  | Bank Identification Number — card prefix; tells issuer + country |
| CNP      | global  | Card Not Present — online card transaction |
| PSP      | global  | Payment Service Provider (Airwallex, Stripe, etc.) |
